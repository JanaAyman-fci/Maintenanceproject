# University Timetabling System (UniTime)

# Project Description
This project is based on the UniTime system, which is used to generate and manage university timetables. It assigns courses, instructors, and rooms into appropriate time slots while avoiding scheduling conflicts.

# Objectives
- Automate timetable generation
- Avoid conflicts between instructors, rooms, and students
- Optimize resource utilization

# Functional Requirements
- Manage courses, rooms, instructors, and students
- Generate schedules automatically
- Detect and resolve conflicts
- Allow manual schedule adjustments

# Non-Functional Requirements
- High performance for large datasets
- Scalability
- Maintainability

## 🔧 Project Scope
This project focuses on maintaining and improving the Java backend of the UniTime system by implementing bug fixes and new features.
